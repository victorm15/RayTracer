# Raytracer

![Main Image](example_img.png)

## Features

* Anti-aliasing
* Shapes
  * Spheres
  * Cubes
* Materials
  * Diffuse/Lambertian
  * Metal (with fuzziness)
  * Dielectrics
* Defocus Blur
* Camera Positioning
* Bounding Volume Hierarchies for Axis-Aligned Bounding Boxes
* Multithreading
